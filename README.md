# XVSqn / EaglesLair

Historical XV Squadron / Tornado GR4 reference and logbook archive project.

This repository is for building the EaglesLair system: a knowledge-led archive, not an OCR-led archive.

## Rule

The uploaded Tornado GR4 reference documents are old and out of date. They are used to design the archive, recognise terms, create controlled fields, and understand historical context. They are not current operational authority.

## Initial architecture

1. Reference Library: SSG, SPG, StanEval, Pocketbook, weapons, TTS and graphics.
2. Knowledge Graph: sorties, systems, weapons, procedures, people, aircraft and places.
3. Logbook Extractor: known form templates, cropped cells, OCR/vision passes and association matching.
4. Review Console: logbook page view with image crop, OCR read, suggested value, confidence and accept/wrong controls.
5. Career Browser: logbook entries linked to training, aircraft, people, documents, photos and memories.
